# 🤖 AI Code Refactoring Tool

This project is an AI-powered tool that analyzes legacy Python code, detects issues, and automatically refactors it into clean, modern, and readable code.

## 🚀 Features
- Detects bad function names
- Identifies formatting issues
- Automatically refactors code
- Simple web interface using Streamlit

## 🛠 Tech Used
- Python
- Streamlit

## ▶️ How to Run
1. Install dependencies:
   pip install streamlit

2. Run the app:
   python -m streamlit run app.py

## 🌐 Deployment
This app can be deployed using Streamlit Community Cloud.
